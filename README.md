# Fraud Detection in Banking Transactions using Random Forest

## Overview

This project focuses on detecting fraudulent banking transactions using a **Random Forest Classifier**. The aim is to build a machine learning model that can distinguish between legitimate and fraudulent transactions based on transaction-related features. The project follows a complete ML workflow, starting from data preprocessing to model evaluation.


## Objectives

* Analyze banking transaction data
* Preprocess and clean the dataset
* Perform exploratory data analysis (EDA)
* Train a Random Forest classification model
* Evaluate the model using standard performance metrics
* Predict whether a transaction is fraudulent or legitimate



## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook



## Workflow

* Imported and explored the dataset
* Handled missing values and encoded categorical features
* Performed feature engineering and scaling where required
* Split the data into training and testing sets
* Trained the model using the Random Forest algorithm
* Evaluated the model using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix
